# Student Result Analyzer 📊

Python project to analyze class results with Pass/Fail logic and Pie Chart visualization using Pandas and Matplotlib.

### Key Insights:
- **Pass Percentage:** 66.7% students passed
- **Class Topper:** Rahul with 91.67%
- **Fail Count:** 2 students below 40%
- **Visualization:** Pie chart for Pass vs Fail distribution

### Tech Stack:
Python, Pandas, Matplotlib

### How to Run:
1. Clone this repo
2. Run `pip install pandas matplotlib`
3. Run `python student_result.py`

### Output:
Generates `result_pie_chart.png` showing pass/fail ratio
